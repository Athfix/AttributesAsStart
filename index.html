  • Attributes

    Well attributes are the one that are inserted in tags which are used for the purpose selection (classes, id, name), styling (style), adding source files (video, audio, images) also adding URL and links Along with it we have value attribute for inputs input type=”values" and values includes text, password, email, radio, range and many thing and source files like source src=”URL” for videos, audio img src=’URL’ for image files and width and height for dimensions width=”100%” and height=”100%”

    Adding the images

    Well to add images in HTML, it must be with tag were it is with attribute of (src) were we are able to add link of image well the extension of image must be either of .jpg, .png, .gif, .jpeg and the image tag here img src=”URL” if image links are to be brought from browsers Just right click on image and select open in new tab and copy the URL and paste it in src, But if the image is local just add the image in the folder were you are writing the HTML file and right click on the image then tap on copy path and paste the path in img src and save the file.
